# Essentials for Obisidian
Remake of the famous Essentials(X) Bukkit/Spigot plugin for Obsidian.

## Commands
### Working commands
| Status | Command | Command description |
|:-:|-|-|
| <ul><li>[x] </li></ul> | essentials | Reloads essentials. |
| <ul><li>[x] </li></ul> | home | Teleport to your home. |
| <ul><li>[x] </li></ul> | gm | Change player gamemode. |

### Work In Progress features
| Status | Command | Command description | Command permission |
|:-:|-|-|
| <ul><li>[ ] </li></ul> | sethome | Set home to your current location. |
| <ul><li>[ ] </li></ul> | delhome | Removes a home. |
